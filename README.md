# Erlang

```erlang
$ sudo apt install erlang
```
```erlang
$ curl -fsSL https://git.io/fpYm4 > mad
$ chmod +x mad
$ sudo cp mad /usr/local/bin
```
```erlang
$ mad dep com pla release cub
$ ./cub parse file priv/mltt.ctt
```
